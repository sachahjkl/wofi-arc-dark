# wofi-arc-dark

A stylesheet recreating the arc-dark theme for wofi.

## Showcase

![Showcase](./pics/showcase.png)

## Install

Simply copy the `style.css` file into the `$XDG_CONFIG_HOME/wofi` folder (so probably `$HOME/.config/wofi`).
